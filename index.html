El examen final consiste en la resolución de dos problemas.

El horario del examen será de 16:00 a 19:00 hs.

Los ejercicios deben entregarse subiendo los códigos al classroom del curso.

El código del primer ejercicio puede ser un Jupyter notebook (ej1_ApellidoNombre.ipynb) o un script de Python (ej1_ApellidoNombre.py).

El segundo ejercicio debe ser entregado como el script de Python, ej2_ApellidoNombre.py.

Análisis de datos

Ejercicio de análisis de datos estadísticos desde una página web. Realizará un pequeño programa para obtener los datos, realizar un análisis preliminar y graficar los resultados.
Análisis de código

Nos han contratado para analizar un problema que ha sido parcialmente resuelto por el cliente. En este ejercicio debe analizar el código del cliente, encontrar los errores y explicar su funcionamiento.
